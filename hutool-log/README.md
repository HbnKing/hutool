## 📚Hutool-log 模块介绍

`Hutool-log`提供对日志门面的封装。

-------------------------------------------------------------------------------

## 🛠️包含内容

### API定义

- `LogEngine`: 日志实现引擎。
- `Log`:       日志对象接口，提供各种级别的日志打印。
- `Level`:     日志级别枚举，提供日志级别定义和规范。
